# Appointment Scheduling System

## 📌 Description

This is a Java-based Appointment Scheduling System developed using Swing for GUI and Maven for project management.

The system allows users to manage appointments, customers, and scheduling operations in an organized and user-friendly way.

---

## 🚀 Features

* Add / Edit / Delete Appointments
* Manage Customers
* Simple and clean GUI (Swing)
* Organized architecture (Model, Service, Repository, UI)
* Unit Testing using JUnit & Mockito
* Test coverage support (JaCoCo)

---

## 🏗️ Project Structure

```
src/main/java/com/team/appointment
├── model        -> Data models
├── repository   -> Data handling
├── service      -> Business logic
├── ui           -> User Interface (Swing)
```

---

## ⚙️ Technologies Used

* Java 8+
* Swing (GUI)
* Maven
* JUnit 5
* Mockito
* JaCoCo (for test coverage)
